About me

Greetings! I'm Priscilla, a passionate web developer specializing in creating visually stunning and user-friendly websites. With a focus on CSS3, HTML5, and vanilla JavaScript, I bring your ideas to life by leveraging the latest technologies and best practices in web development.

Skills:

CSS3:

I create captivating designs and layouts for websites. I utilize CSS features like flexbox, grid, transitions, and animations to enhance the visual appeal and interactivity of user experiences.

HTML5:

Well-versed in HTML5, I develop clean and well-structured HTML code to ensure optimal website performance and compatibility across different devices and browsers. I prioritize semantic markup and accessibility principles.

Vanilla JavaScript:

With a strong command of vanilla JavaScript, I build interactive and dynamic elements for websites, enhancing user interactions and enabling data manipulation. JavaScript's powerful capabilities empower me to create engaging experiences.

Bootstrap:

Experienced in Bootstrap, a popular front-end framework, I rapidly prototype and develop responsive websites. By leveraging Bootstrap's pre-built components and grid system, I create mobile-first and fully responsive designs.

Live API Integration:

Skilled in integrating live APIs into web applications, I enable real-time data retrieval and updates. Whether it's fetching weather data or social media feeds, I seamlessly integrate APIs to enhance website functionality and deliver a dynamic user experience.

JS Moment:

Proficient in the JavaScript library Moment.js, I simplify handling dates, times, and time zones. Utilizing Moment.js, I effortlessly format, manipulate, and display dates and times on websites, ensuring a seamless user experience across different time zones.

Responsive Design:

Prioritizing responsive design, I create websites that adapt flawlessly to various screen sizes. By utilizing CSS media queries and responsive design principles, I ensure optimal viewing and functionality on desktops, tablets, and smartphones.

I thrive on challenging projects that allow me to push the boundaries of my creativity and problem-solving abilities.
